Autism Spectrum Disorder Prediction Project

This project utilizes a machine learning model to predict the likelihood of a person being on the autism spectrum. The model provides a simple yet clear prediction and is designed to be used for children aged between 2 to 14 years old.
Description

This project is aimed at predicting the possibility of a child being on the autism spectrum disorder (ASD). By leveraging a pre-trained machine learning model, the project delivers straightforward predictions without the need for a database. Simply clone the project and follow the provided steps to get started.
Features

    Machine learning-based prediction of autism spectrum disorder.
    Designed specifically for children aged 2 to 14 years old.
